Observatorio Comida Rapida
==============

Projecto Django
Observatorio Comida Rapida usando la api de twitter

Pasos:
* Crear directorio: mkdir bd
* acceder: cd bd
* pip install django
* git init
* git clone https://github.com/luisriq/ObservatorioFF.git
* ./manage.py runserver
* Acceder a localhost:8000/index/
Requerimientos: 

* Django 1.7
* python 2.7
* Tweepy
* Postgres
