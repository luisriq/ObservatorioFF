ObservatorioFF
==============

Projecto Django, observatorio Comida Rapida usando la api de twitter

Pasos:
* Crear directorio: mkdir bd
* acceder: cd bd
* pip install django
* git init
* git clone https://github.com/luisriq/ObservatorioFF.git
* ./manage.py runserver

acceder a localhost:8000
o localhost:8000/home

Versiones: 

* Django 1.7

* python 2.7
